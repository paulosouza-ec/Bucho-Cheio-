# Estrutura do projeto:

### Dependências

- React
- Node 
- yarn
- Docker

### Instalando dependências
```
cd front
yarn install
cd ..
cd server
sudo yarn install
```
### Rodando o projeto
Iniciar o servidor
```
cd server
docker-compose up
```
Iniciando o frontend
```
cd front
yarn start
```
